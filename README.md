Installation via Packagist
--------------------------
```
composer create-project delboy1978uk/palindrome
```
Testing with Codeception
------------------------
```
vendor/bin/codecept run unit --coverage-html
```
Travis
------
Tests are being run on PHP 5.4, 5.5, 5.6, and HHVM
